## 🚀 How to Use

You can run these files in any supported IDE or using the terminal.

### For Java:
```bash
javac FileName.java
java FileName
```
For C:
```
gcc file_name.c -o output
./output
```

📌 Each file includes comments explaining the logic and approach.
