This repository contains my solutions to various **LeetCode coding problems**, solved using **Java**, **C**, and **MySQL**. The goal is to improve problem-solving skills, strengthen coding fundamentals, and prepare for technical interviews.

- 🔁 Build consistency in solving coding problems

- 🔄 Practice writing clean and efficient code in Java, C, and SQL queries
 
---

## 🚀 How to Use

You can run these files in any supported IDE or using the terminal.

### For Java:
```bash
javac FileName.java
java FileName
```
### For C:
```bash
gcc file_name.c -o output
./output
```

### For MySQL:
```bash
mysql -u username -p database_name < file_name.sql
```
Or run directly in MySQL Workbench, phpMyAdmin, or any SQL client.

📌 Each file includes comments explaining the logic and approach.
